# Inventory-Dashboard📦


A responsive **Inventory Dashboard** built with **HTML, CSS, and JavaScript**.  
It provides an overview of inventory operations including products, suppliers, and orders, with interactive charts powered by **Chart.js**.


## 🚀 Features
- Sidebar navigation for quick access to sections
- Summary cards showing:
  - Total Products
  - Suppliers
  - Pending Orders
- Interactive bar chart for inventory statistics
- Recent inventory activity table
- Responsive design for desktop and mobile
- Easy to extend with more charts or modules


## 🛠️ Tech Stack
- **HTML5** – structure
- **CSS3** – styling
- **JavaScript (ES6)** – interactivity
- **Chart.js** – data visualization
